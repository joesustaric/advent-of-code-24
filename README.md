# Advent of Code 2024

My attempt to try and solve a few of these puzzles. https://adventofcode.com/


## Setup

Requires Python 3.11.4 then install [Poetry](https://python-poetry.org/):

```bash
pipx install poetry
```

Then install the dependencies:

```bash
poetry install
```

To run the tests:

```bash
poetry run pytest
```
